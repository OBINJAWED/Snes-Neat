# Snes-Neat

## Overview

Train a neural network to play a video game using raw pixel data. 
The Gym Retro platform includes an emulator for the Super Nintendo. The emulator used is
SNES9x. 

The agent receives a reward for each frame that it moves right.


## Requirements:

• Python 3 +

• OpenAI Gym

• Gym Retro

• Tensorflow

• Matplotlib

• Gym Integration Tool

• Super Mario World Game


## Results

The agent was able to complete the level in about 4 hours of training.
